# MIRAI ARXR - AR Smart Glasses for Heritage Tourism

## Project Overview
MIRAI ARXR is a web-based presentation platform for cutting-edge AR smart glasses developed to transform heritage tourism in India. The glasses provide immersive historical and cultural experiences by overlaying 3D augmented reality reconstructions on heritage sites, supported by multilingual AI storytelling. Their offline-first architecture and affordable hardware design aim to break barriers of cost, language, and connectivity traditionally limiting tourism accessibility.

## Features & Benefits

### Product Highlights
- **3D AR Reconstruction:** Real-time, photo-realistic AR overlay of monuments and heritage sites.
- **Multilingual AI Narration:** Support for 10+ Indian languages with dynamic, location-based storytelling.
- **Offline Operation:** Edge AI and storage enable fully offline usage with no internet dependency.
- **Sustainable Power:** Solar-powered rechargeable batteries for all-day usage during tours.
- **Intuitive Interaction:** Gesture and voice commands for easy navigation and content access.
- **Privacy by Design:** All data processing is done on the device ensuring zero cloud data exposure.

- **Frontend:** HTML5, CSS3 using Google Fonts (Poppins) and Flexbox for layout.
- **Interactivity:** JavaScript for video swiper and smooth navigation effects.
- **Design:** Minimalistic, user-friendly, with emphasis on readability and accessibility.
